# مولّد رمز WPS PIN - بدون إنترنت

أداة بسيطة بوجهة عربية تُولّد رمز WPS PIN المحتمل من عنوان MAC (BSSID) الخاص بنقطة الوصول. تعمل بشكل كامل دون اتصال بالإنترنت، وتدعم خوارزميات أجهزة راوتر شهيرة.

## الميزات
- واجهة رسومية سهلة الاستخدام باللغة العربية.
- جلب تلقائي لعنوان MAC للشبكة المتصل بها حالياً (ويندوز / لينكس / ماك).
- يدعم خوارزميات: D-Link, Thomson/SpeedTouch, Arcadyan, Pirelli.
- حساب خانة التحقق Checksum تلقائياً.
- نسخ النتائج إلى الحافظة بنقرة واحدة.
- لا يحتاج لأي اتصال بالإنترنت (يعمل أوفلاين).
